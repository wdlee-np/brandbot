# BUG-10-03: AI 자동생성 퀴즈 정답 10자 제한으로 잘림

## 현상
- AI 자동 생성 퀴즈의 정답이 10자를 초과할 경우 강제로 10자까지만 저장됨
- 예: "미국 조지아주 애틀랜타" → "미국 조지아주" 로 잘려 저장

## 근본 원인
- `services/quiz/generate.ts`: `q.answer.slice(0, 10)` 강제 절삭
- `ai/prompts/quiz-generate.txt`: "10자 이내" 제약 명시
- `QuizEditor.tsx`: maxLength=10, 10자 초과 validation
- `app/api/admin/projects/[id]/quizzes/route.ts`: answer 10자 초과 거부
- `app/api/admin/projects/[id]/quizzes/[quizId]/route.ts`: answer 10자 초과 거부
- `supabase/migrations/001_initial_schema.sql`: DB CHECK `char_length(answer) <= 10`

## 수정
- 모든 10자 제한을 20자로 변경
- DB 마이그레이션(`002_extend_limits.sql`)으로 CHECK constraint 변경
