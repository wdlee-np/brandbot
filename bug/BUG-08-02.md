# BUG-08-02: AI가 DB 등록 퀴즈 범위를 벗어나 자유롭게 퀴즈를 생성

## 증상
AI 챗봇이 DB에 등록된 퀴즈 문제를 사용하지 않고 임의로 퀴즈를 만들어 출제하는 경우가 발생.
또한 퀴즈나 쿠폰 시스템에 대한 사용자 질문에 응답을 거부하는 문제도 포함.

## 원인
1. `ai/prompts/brand-system.txt`의 퀴즈 관련 규칙이 불충분:
   - 힌트 제공 시 정답 단어 포함 금지 규칙 없음
   - 퀴즈/쿠폰 시스템 설명 허용 규칙 없음
   - 사용자가 퀴즈를 요청할 때 즉시 제공해야 한다는 규칙 없음
2. `app/api/chat/[projectId]/message/route.ts`에서 AI 응답에 `[QUIZ_STEP:N]` 마커가 있을 때 DB 퀴즈 문제가 응답에 없으면 추가하는 포스트프로세싱 없음

## 수정 방법
1. `brand-system.txt`에 아래 규칙 추가:
   - 힌트 시 정답 단어 절대 사용 금지 규칙
   - 퀴즈/쿠폰 시스템 설명 허용 규칙
   - 사용자 퀴즈 요청 시 즉시 제공 규칙
2. `message/route.ts`에서 AI 응답 포스트프로세싱: `[QUIZ_STEP:N]` 감지 시 DB 퀴즈 문제가 응답에 포함되지 않았다면 문제를 강제 추가

## 영향 범위
- `ai/prompts/brand-system.txt`
- `app/api/chat/[projectId]/message/route.ts`
