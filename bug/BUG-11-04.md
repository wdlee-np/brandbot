# BUG-11-04: 채팅창 키보드 포커스 자동 유지

## 현상
채팅창 진입 후 텍스트 입력창에 포커스가 자동으로 맞춰지지 않음.
봇 응답 후에도 포커스가 해제되어 매번 수동으로 탭/클릭해야 함.

## 수정 내용
`ChatInput.tsx`에 다음 처리 추가:
1. 초기 마운트 시 `autoFocus` 적용
2. `disabled` prop이 `true → false`로 바뀔 때(봇 응답 완료 시) 자동 포커스 복귀
3. 메시지 전송 후 즉시 포커스 복귀

## 수정 파일
