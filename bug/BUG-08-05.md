# BUG-08-05: 힌트 제공 시 정답 단어가 포함된 표현 사용

## 증상
AI가 퀴즈 힌트를 줄 때 정답 단어 자체 또는 정답이 포함된 표현을 사용하여 사실상 정답을 노출하는 경우 발생.

## 원인
`ai/prompts/brand-system.txt`의 규칙 5번(퀴즈 정답 노출 금지)이 힌트 제공 시나리오를 명확히 다루지 않음.
서버에서 AI에게 전달하는 힌트 지시어에도 정답 단어 사용 금지 명시가 없음.

## 수정 방법
1. `brand-system.txt` 규칙 5번 강화:
   - "힌트 제공 시 정답 단어와 동일하거나 정답을 포함하는 단어/표현 절대 사용 금지"
   - "간접적인 힌트(속성, 특징, 연상 단어)만 허용"
2. `message/route.ts`의 힌트 지시어에 명시적 제약 포함:
   - `"정답 단어나 정답이 포함된 표현 절대 금지. 간접 힌트만."`

## 영향 범위
- `ai/prompts/brand-system.txt`
- `app/api/chat/[projectId]/message/route.ts` (힌트 지시어 문구)
