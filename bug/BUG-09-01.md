# BUG-09-01: 퀴즈 정답 시 동일 메시지에 두 개의 퀴즈 문제 노출

## 현상
- 퀴즈를 정답 맞혔을 때, AI 응답에 축하 메시지와 함께 다음 퀴즈 문제가 2개 포함됨
- 예: "정답입니다! 🎉 [다음 퀴즈 출제 내용] \n\n [서버 후처리로 추가된 동일 또는 엉뚱한 퀴즈 문제]"
- 이유: `quizAnswerContext` 모드(정답 판정 중)에도 서버 후처리가 조건 없이 실행됨

## 근본 원인
`app/api/chat/[projectId]/message/route.ts`의 post-processing 로직:
```typescript
// 244줄: quizAnswerContext 유무와 관계없이 무조건 실행
if (quizStep !== null && currentQuiz && !cleanText.includes(currentQuiz.question)) {
  finalText = `${cleanText}\n\n${currentQuiz.question}`;
}
```
- `quizAnswerContext`가 있는 경우(정답/오답 판정 중)에는 이 후처리를 건너뛰어야 함
- 또한 정답 축하 메시지에서 AI가 다음 퀴즈를 자발적으로 출제하지 않도록 지시가 불충분함

## 수정 방향
1. post-processing 조건에 `!quizAnswerContext` 추가
2. 정답/강제통과 `quizInstruction`에 "다음 퀴즈 출제 금지" 명시적 지시 추가

## 영향 파일
- `app/api/chat/[projectId]/message/route.ts`
