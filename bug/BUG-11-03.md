# BUG-11-03: 퀴즈 완료 시 별도 쿠폰 페이지 대신 채팅창 내 쿠폰 제공

## 현상
퀴즈를 모두 통과하면 채팅창을 벗어나 `/chat/[projectId]/reward` 페이지로 리다이렉트됨.
사용자 경험이 끊기고 채팅 맥락이 사라짐.

## 수정 내용
- `ChatPageClient.tsx`: `onQuizComplete` 라우터 리다이렉트 제거
- `ChatWindow.tsx`: 퀴즈 완료 감지 시 `/api/chat/[projectId]/coupon` API 호출 후 결과를 채팅창 내 봇 메시지 스타일의 쿠폰 카드로 표시
- `app/api/chat/[projectId]/coupon/route.ts`: quiz_progress 완료 판정을 실제 퀴즈 수 기준으로 수정 (기존 하드코딩 `< 3` → DB 실제 count)

## 수정 파일
- `components/chat/ChatPageClient.tsx`
- `components/chat/ChatWindow.tsx`
- `app/api/chat/[projectId]/coupon/route.ts`
