# BUG-08-01: OfficialQuizCard 팝업이 갑자기 나타남

## 증상
채팅 화면에서 퀴즈 모드가 활성화될 때 `OfficialQuizCard` 컴포넌트가 메시지 목록 아래에 갑자기 팝업처럼 나타나 사용자 경험을 해침.

## 원인
`components/chat/ChatWindow.tsx`의 164번째 줄:
```tsx
{isQuizMode && currentQuizStep !== null && currentQuizQuestion && (
  <OfficialQuizCard step={currentQuizStep} question={currentQuizQuestion} />
)}
```
AI 응답 메시지에 이미 퀴즈 문제가 포함되어 있음에도 불구하고, 별도의 카드 컴포넌트를 추가로 표시하여 중복 표시 및 UX 저하 발생.

## 수정 방법
`ChatWindow.tsx`에서 `OfficialQuizCard` 컴포넌트 정의와 렌더링 코드를 모두 제거.
AI 응답 메시지 안에 퀴즈 문제가 이미 포함되어 있으므로 별도 카드 불필요.

## 영향 범위
- `components/chat/ChatWindow.tsx`
