# BUG-08-03: 어드민 퀴즈 편집/삭제 API 라우트 누락

## 증상
어드민 패널의 `QuizEditor` 컴포넌트에서 개별 퀴즈 편집(PATCH) 및 삭제(DELETE) 버튼 클릭 시 404 오류 발생.

## 원인
`components/admin/QuizEditor.tsx`에서 아래 엔드포인트를 호출하지만:
- `PATCH /api/admin/projects/${projectId}/quizzes/${quizId}`
- `DELETE /api/admin/projects/${projectId}/quizzes/${quizId}`

해당 라우트 파일 `app/api/admin/projects/[id]/quizzes/[quizId]/route.ts`가 존재하지 않음.
현재 `/quizzes/route.ts`에는 GET과 POST만 구현되어 있음.

## 수정 방법
`app/api/admin/projects/[id]/quizzes/[quizId]/route.ts` 파일 생성:
- `PATCH`: `question`, `answer` 필드 수정 (answer 10자 이하 검증)
- `DELETE`: 퀴즈 삭제
- 두 엔드포인트 모두 `requireAdmin()` 인증 체크 포함

## 영향 범위
- `app/api/admin/projects/[id]/quizzes/[quizId]/route.ts` (신규 생성)
