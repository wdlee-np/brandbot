# BUG-10-02: Flash Lite 외 Gemini 모델 선택 시 오류

## 현상
- 프로젝트 생성 또는 상세 페이지에서 Flash Lite 외 모델 선택 후 채팅/생성 시 API 오류 발생
- `gemini-2.5-flash`, `gemini-3.1-pro-preview` 모두 작동하지 않음

## 근본 원인
1. `gemini-2.5-flash`: 2025년 이전 버전으로 2026년 기준 deprecated 또는 이름 변경됨
2. `gemini-3.1-pro-preview`: API에서 지원하지 않는 정확하지 않은 모델 ID
3. `services/gemini/client.ts`의 fallback 모델도 `gemini-2.5-flash`로 하드코딩되어 있어 폴백도 실패

## 수정
- 현재 작동하는 `gemini-3.1-flash-lite-preview` 패턴 기반으로 모델명 정비
- `gemini-2.5-flash` → `gemini-3.1-flash-preview` (동일 버전 계열 표준 Flash)
- fallback 모델을 `gemini-2.5-flash` 하드코딩 → `DEFAULT_MODEL`로 변경
- 라벨에 정확한 모델 ID 포함하여 표시
